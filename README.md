# Travel Planning Agent 智能旅行规划助手

这是一个基于大语言模型（LLM）的智能旅行规划助手，通过**ReAct**（Reasoning + Acting）模式与用户交互，动态调用外部工具获取实时数据，最终生成详细的行程计划。

## ✨ 功能特点

- **天气查询** – 使用高德地图天气 API 获取指定城市的实时天气。
- **景点搜索** – 利用 Tavily 搜索引擎根据城市和天气推荐景点。
- **住宿推荐** – 基于城市、景点、天气和用户偏好，由 LLM 生成住宿区域建议。
- **地理编码** – 查询景点、地铁站等地点的高德坐标（经纬度）。
- **行程耗时矩阵** – 计算多个地点之间的驾车耗时，用于智能分组。
- **每日行程分组** – 基于耗时矩阵，将景点按天自动分组（贪心算法）。
- **路径规划** – 获取两点之间的交通建议（支持公交、驾车、步行、骑行，公交失败可自动回退到驾车）。
- **偏好记忆** – 记录用户的目的地、天数、兴趣等，供后续工具使用。
- **交互式问答** – 自动向用户提问以收集缺失信息（如出发日期、预算等）。
- **完整行程输出** – 最终生成包含每日安排、交通方式、温馨提示的旅行计划。

## 📦 安装与依赖

### 环境要求
- Python 3.8 或更高版本
- 建议使用虚拟环境（如 `venv` 或 `conda`）

### 安装依赖包
```bash
pip install requests tavily-python openai
```
> `json`、`re`、`os` 等为 Python 标准库，无需额外安装。

## 🔑 配置 API 密钥

本程序需要三个外部服务的 API 密钥：

1. **高德地图 API Key** – 用于天气查询、地理编码、距离矩阵、路径规划  
   - 申请地址：[高德开放平台](https://lbs.amap.com/)  
   - 需要开通 **Web 服务 API**（天气、地理编码、距离测量、路径规划）

2. **Tavily API Key** – 用于景点搜索  
   - 申请地址：[Tavily](https://tavily.com/)  
   - 免费版有一定调用额度

3. **兼容 OpenAI 的 LLM API Key** – 用于驱动智能体的思考与工具调用  
   - 例如 [OpenAI](https://platform.openai.com/)、[AIHubMix](https://aihubmix.com/) 等  
   - 需要提供 `api_key`、`base_url` 和 `model` 名称

### 设置密钥的方式

在代码最后的主执行单元格中（`# --- 1. 配置LLM客户端 ---` 部分），直接修改变量值：

```python
API_KEY = "your-llm-api-key"
BASE_URL = "https://api.example.com/v1"   # 根据服务商修改
MODEL_ID = "your-model-name"

# 高德和 Tavily 的密钥也可通过环境变量设置
os.environ['TAVILY_API_KEY'] = "your-tavily-key"
os.environ['GAODE_API_KEY'] = "your-gaode-key"
```

> ⚠️ **安全提示**：请勿将密钥硬编码后提交到公开仓库。推荐使用环境变量或配置文件管理。

## 🚀 使用方法

1. **启动 Jupyter Notebook**  
   ```bash
   jupyter notebook travel_agent.ipynb
   ```

2. **依次运行所有单元格**，直到最后一个包含主循环的单元格。

3. **程序会提示**：
   ```
   请输入您的旅行需求（例如：我想去北京玩3天，喜欢历史文化）：
   ```
   输入你的需求，例如：
   ```
   我想去上海玩3天，喜欢自然风光
   ```

4. **智能体将自动进行多轮交互**：
   - 可能询问出发日期、预算等补充信息。
   - 调用天气、景点、住宿推荐等工具。
   - 记录偏好，最终生成行程计划。

5. **最终输出**将包含详细的每日行程、住宿推荐和温馨提示。
